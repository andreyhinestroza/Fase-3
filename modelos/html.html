    GIT y GITHUB


    Git Es un software gratuito y de código abierto como Linux, que brinda un sistema de control de versiones muy útil para el programador ya que permite manejar el proyecto de manera rápida, eficaz y puede mitigar errores en el proceso herramienta muy importante para programar y así no perder lo realizado, este brinda tres estados para el control de versiones, comited: en este estado los datos se encuentran ya almacenados en una base de datos local, modified: aquí haz modificado datos pero no se han incluido en la base de datos local, staged: en este se ha marcado un archivo modificado en su versión actual para entrar al siguiente commit. Este Git se trabaja de carácter local desde tu pc o dispositivo que utilices, con el objeto de poder subirlo a red global a través de github para ser compartido. GitHub es un sitio web social o comunitario donde los usuarios trabajan sobre un repositorio donde aprenden, comparte y trabajan en grupo para construir el software que se propone a realizar, el link es: https://github.com
